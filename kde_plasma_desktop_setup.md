# KDE Plasma + Essential Apps (Arch Linux)

## Install KDE Plasma (minimal desktop)

```bash
sudo pacman -S plasma
```

---

## Install essential desktop apps

```bash
sudo pacman -S dolphin konsole kate chromium htop okular spectacle ark klipper
```

---

## What this includes

* **Dolphin** → File manager
* **Konsole** → Terminal
* **Kate** → Text editor
* **Chromium** → Web browser
* **htop** → System monitor
* **Okular** → PDF viewer
* **Spectacle** → Screenshots
* **Ark** → Archive manager (zip/rar/etc)
* **Klipper** → Clipboard history manager

---

## Notes

* `plasma` installs the base KDE desktop environment only
* The apps above complete a usable daily desktop setup
* Everything comes from official Arch repositories (no AUR required)
